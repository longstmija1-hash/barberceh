# barberceh

Современный сайт барбершопа **ЦЕХ** (Екатеринбург) на Next.js.

## Стек

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS + Shadcn UI
- Framer Motion + GSAP
- Yclients (онлайн-запись)

## Запуск

```bash
npm install
npm run dev
```

Открой [http://localhost:3000](http://localhost:3000).

## Сборка

```bash
npm run build
npm start
```

## Маршруты

- `/` — главная
- `/uslugi/` — услуги и цены
- `/mastera-cexa/` — мастера
- `/novosti/` — новости
- `/magazin/` — сертификаты
- `/kontakty/` — контакты
